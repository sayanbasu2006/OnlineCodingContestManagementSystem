# MVP Backend Description (For Presentation)

* **Core Purpose:** The MVP backend provides a functional RESTful API architecture using Node.js, Express, and MySQL to support essential coding contest features.
* **Authentication:** Implemented secure User Management with JWT-based authentication and role-based access control for Admins.
* **Contests & Problems:** Admins can seamlessly create and manage coding contests alongside specific coding problems assigned to them.
* **Participation:** Users can actively join contests and submit coding solutions to individual problems natively verified through endpoints.
* **Leaderboard Logic:** The system processes and aggregates submission data dynamically to generate real-time global and contest-specific leaderboards.
